# Base de Datos - Tecnica 1

Proyecto de la materia de Base de Datos del profesor Sosa (Escuela Tecnica 1).

Este repo esta pensado para aprender haciendo: modelar, cargar, consultar y mejorar bases de datos relacionales y no relacionales, paso a paso.

## Objetivo

Aprender fundamentos y buenas practicas para:

- Disenar estructuras de datos claras y mantenibles.
- Trabajar con SQL y consultas utiles para problemas reales.
- Comprender diferencias entre modelos relacionales y no relacionales.
- Documentar decisiones tecnicas como en un entorno profesional.

## Estructura del proyecto

- `tablas/`: scripts de creacion y definicion de tablas/esquemas.
- `datos/`: inserts, cargas iniciales y datos de prueba.
- `procedimientos/`: procedimientos almacenados, funciones y logica reutilizable.

## Forma de trabajo

- Empezar simple.
- Probar todo con datos reales o de ejemplo.
- Versionar cada cambio importante.
- Dejar cada consulta mas clara de como la encontramos.

## Idea fuerza

No se trata solo de que "funcione": se trata de entender por que funciona,
como escalarlo y como mantenerlo.

Aprender base de datos es aprender a pensar sistemas.
